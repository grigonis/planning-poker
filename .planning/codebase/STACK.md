# Technology Stack

## Frontend
- **Framework**: [React 18](https://react.dev/) (Vite)
- **Styling**: [Tailwind CSS 3.4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Real-time**: [Socket.io-client](https://socket.io/)
- **Routing**: [React Router 7](https://reactrouter.com/)
- **Shaders**: `@paper-design/shaders-react`
- **Avatars**: `@dicebear/core` + `@dicebear/collection`

## Backend
- **Runtime**: [Node.js](https://nodejs.org/)
- **Framework**: [Express 5](https://expressjs.com/) (Next-gen version)
- **WebSockets**: [Socket.io](https://socket.io/)
- **Utilities**: `uuid`, `cors`, `dotenv`
- **Process Management**: `nodemon` (dev)

## Configuration
- **Vite**: `client/vite.config.js`
- **PostCSS/Tailwind**: `client/postcss.config.js`, `client/tailwind.config.js`
- **ESLint**: `client/.eslintrc.cjs`
- **Environment**: `.env` (server)
