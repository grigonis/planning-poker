# T01: Create tsconfig.json, wire Vite alias, and run shadcn init

**Created tsconfig.json path alias and Vite resolve alias, then ran shadcn init with the nova preset to scaffold components.json, utils.ts, and button.tsx.**

## What Happened

1. Created `client/tsconfig.json` with `compilerOptions.baseUrl: "."` and `paths: { "@/*": ["./src/*"] }`.
2. Patched `client/vite.config.js` to import `path` and add `resolve.alias: { "@": path.resolve(__dirname, "./src") }`.
3. Ran `npx shadcn@latest init --preset nova --template vite --yes` from `client/`. All preflight checks passed (Vite framework detected, Tailwind CSS validated, import alias validated). Generated: `src/components/ui/button.tsx`, `src/lib/utils.ts`, updated `src/index.css`, wrote `components.json`.
4. Ran `npm install` — completed clean (up to date, no new deps needed beyond what init installed inline).
