# Keystimate

**Elevator Pitch:** A premium, real-time Planning Poker application designed for remote agile teams who value aesthetics and seamless user experience.

## Core Value
- **Aesthetic Excellence:** Modern UI with smooth animations and dark mode.
- **Real-time Feedback:** Instant voting synchronization and live group management.
- **Team Workspace:** Persistent user identities and session history tracking.

## Current State
Keystimate is now a complete Team Workspace (M006 complete). The application features a persistent Dashboard where users can:
1. Manage their global profile (name, avatar).
2. Review a detailed history of all past voting sessions (archived in-memory).
3. Track and rejoin multiple active workspaces.
4. Create new estimation rooms instantly.

Next steps will involve transitioning from in-memory and `localStorage` persistence to a full Database integration (M007).
